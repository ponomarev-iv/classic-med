Электронейростимуляция

Применяется для стимуляции нервов и мышц. При двигательных нарушениях в виде параличей, парезов, после перенесённых инсультов, травм нервной системы, невралгиях, невритах.

Широко используется для коррекции мышечного корсета. Для релаксации мышц с целью снятия напряжения, усталости. Или для повышения тонуса ослабленных мышц.
